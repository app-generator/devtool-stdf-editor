# STDF Edit Tool

Show SDTF as table, and support to split/merge/edit STDF file. 

<br />

> Usage:

Create & activate a VENV and after run: 

```bash
$ pip install -r requirements.txt
$ python sample.py                # CLI version
$ python sample-ui.py             # UI Version
```
